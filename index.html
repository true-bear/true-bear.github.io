🔹 상세 경력 기술 이력

  • 크롤러 서버는 goroutine 기반으로 구현하여 체인 블록을 주기적으로 긁고, 트랜잭션 상태를 파악해 DB 업데이트
  • 체인 데이터는 IPFS와 연동되어 있으며, 콘텐츠(예: NFT 이미지, 메타데이터)는 미리 가공 및 저장
  • 게임과 연결된 재화(Tx 기반 HBI 토큰 등)는 블록 컨펌 후 인벤토리에 반영되며, 클라이언트는 API 호출만으로 즉시 확인 가능
  • 인증 처리 시, refresh token은 Redis 기반으로 관리하여 세션 충돌 최소화 및 다중 로그인 대응
  • 대시보드 및 운영툴 API도 모듈화하여, 관리자 전용 기능(아이템 지급, Tx 모니터링 등)을 라우팅 기반으로 처리
  • 대부분의 서버는 공통 유틸과 인터페이스 기반으로 작성되어 신규 기능 추가 시 소스 코드 파편화 없이 통합 가능
